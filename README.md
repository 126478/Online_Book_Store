# Book Store
An online eCommerce web store, built with ASP.NET MVC in C#.
## Technologies used
+ ASP.NET Core MVC
+ Entity Framework Core 6.0
# Getting Started
## Prerequisites
+ Visual Studio 2022
+ .NET 6
## Installation
1.Clone the repo
```
https://github.com/126478/Online_Book_Store.git
```
2.Configure connection string in appsettings.json
